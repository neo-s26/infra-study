# 📅 Study Plan (半年ロードマップ)

この `study-plan.md` は、インフラ／クラウド／自動化を最優先しつつ、基本情報技術者試験（午前・午後）もゆるく並行するための計画書です。
在宅勤務のスキマ時間でも進められるように、短いタスク単位で構成しています。

※勉強開始は2026年～
　それまではネットショップ検定の勉強を進める

---

# 🧭 全体方針

* **インフラ・クラウド（Azure → AWS）を最優先**
* **基本情報は“午前アプリ中心＋午後は重点科目のみ”で並行**
* GitHub を使って学習ログとメモを管理
* 少しずつ積み上げて半年で大きく伸ばす

---

# 📆 1〜2ヶ月目：Azure基礎（AZ-900）

## 🎯 目標

* Azure の基本概念を理解
* VM、ストレージ、ネットワークの概要を説明できる
* 基本情報 午前問題に慣れる

## 🔧 やること

* Microsoft Learn の AZ-900 基礎コースをすすめる
* Azure ポータルで VM / Storage / VNet を触ってみる
* GitHub に daily log を書く習慣をつける
* 基本情報 午前問題アプリを毎日 10〜20問

---

# 📆 3〜4ヶ月目：Azure 管理（AZ-104）

## 🎯 目標

* Azure 管理レベルの知識（VM、VNet、NSG、Monitor）を持つ
* PowerShell の基礎を理解
* 基本情報 午後の「ネットワーク」「データベース」をクリア

## 🔧 やること

* VM 構築 → NSG → モニタリングを一通り練習
* Azure Monitor / Log Analytics を触る
* PowerShell 基本コマンド（Get-Service / イベントログ取得等）
* 基本情報 午後：ネットワークとデータベースだけ重点学習

---

# 📆 5〜6ヶ月目：AWS基礎（CLF）＋自動化（PowerShell/Python）

## 🎯 目標

* AWS の基本サービスを理解（EC2 / S3 / IAM / VPC）
* Azure との比較ができる
* 小規模な自動化スクリプトを実装
* 基本情報はいつ受けても合格圏

## 🔧 やること

* AWS Cloud Practitioner の学習
* EC2 起動・S3 バケット作成・IAM 設定の演習
* PowerShell で簡単な自動化スクリプト作成
* 余力があれば Python も着手
* 基本情報は過去問道場で総仕上げ

---

# 📚 推奨教材まとめ

* **Microsoft Learn (Azure 基礎〜管理)**
* **Udemy（AZ-104 / AWS CLF）**
* **GitHub：daily-log とメモ管理**
* **基本情報 午前：スマホアプリ / 過去問道場**
* **基本情報 午後：ネットワーク / データベース重点**
