# 📅 Study Plan (半年ロードマップ)

この `study-plan.md` は、インフラ／クラウド／を最優先しつつ、基本情報技術者試験（午前・午後）もゆるく並行するための計画書です。
在宅勤務のスキマ時間でも進められるように、短いタスク単位で構成しています。

※勉強開始は2026年～

---

# 🧭 全体方針

* 基本的には参考書を読み上げオーディオ化し、業務中等に聞く→休日に試験対策を行う運用
* **インフラ・クラウド（Azure → AWS）を最優先**
* **基本情報は“午前アプリ中心＋午後はAzure取得後重点科目のみ”で並行**
* GitHub を使って**試験型ガイドやAzure→AWSやJava→C＃の読み替え**等を管理
* 暗記や過去問を解く際は、iPadのOneNoteアプリで管理
* 体調によりスケジュールは前後してOK（期限は目安）
